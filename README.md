# Landing Page Personal de Servicios

## Descripción

Este proyecto es una landing page profesional diseñada para presentar y promocionar los servicios de Raldy Lopez. Combina un diseño moderno y minimalista con una estructura clara y responsiva, optimizada para atraer clientes y mostrar credenciales de manera efectiva.

## Características

- Diseño limpio y elegante con enfoque minimalista
- Estructura HTML semántica y CSS personalizado
- Totalmente responsivo y compatible con dispositivos móviles
- Secciones organizadas para destacar servicios, contacto y branding personal
- Despliegue público mediante GitHub Pages
